# Retrieval Chain Trainer

A mobile-friendly static web app for turning articles into English retrieval practice.

## How it works

1. Paste an article into your Custom GPT / ChatGPT.
2. Ask it to output Retrieval Pack JSON.
3. Paste the JSON into this app.
4. Practice chains, collocations, why/how questions, speaking prompts, and summary recall.

No OpenAI API key is required. The AI step happens manually in ChatGPT.

## GitHub Pages deployment

1. Create a new GitHub repository.
2. Upload these files:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `README.md`
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch `main` and folder `/root`.
6. Open the Pages URL on your phone.

## Custom GPT instruction

Inside the app, tap **Copy Custom GPT Prompt** and paste it into your Custom GPT instructions.

## Local testing

Just open `index.html` in your browser.
