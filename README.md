# Retrieval Trainer V2

A mobile-first static web app for English retrieval practice.

## Features

- Paste Retrieval Pack JSON generated by your Custom GPT
- Save packs into a topic-based library
- Chain Recall practice
- Collocation Recall practice
- Why/How question practice
- Summary Recall
- Review wrong answers
- LocalStorage persistence
- GitHub Pages ready

## How to deploy on GitHub Pages

1. Create a new GitHub repository.
2. Upload these files to the repository root:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `README.md`
3. Go to Settings → Pages.
4. Source: Deploy from branch.
5. Branch: main, Folder: / root.
6. Save and open the GitHub Pages URL.

## Custom GPT prompt

Convert this article into a Retrieval Pack JSON for my app. Output valid JSON only. Include title, topic, 8-12 chains, 15-25 collocations, why_questions, how_questions, and summary_task. Each chain should follow Concept → Mechanism → Immediate Effect → Outcome → Long-term Consequence.
